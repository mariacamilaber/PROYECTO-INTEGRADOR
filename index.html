Descripcion del proyecto

Este sistema está enfocado en identificar posibles riesgos en los estudiantes para mejorar su rendimiento y bienestar

¿Por qué es importante?

AURA centraliza la información estudiantil y permite detectar problemas de forma temprana.


Objetivo

El objetivo principal es desarrollar un prototipo funcional que integre un sistema en Java para el análisis de datos y una interfaz web que permita ingresar y visualizar la información de los estudiantes.


¿Cómo funciona el sistema?

  1. El estudiante llena el formulario
  2. Los datos se almacenan en matrices
  3. Java procesa la información
  4. Se clasifica el riesgo
  5. Se muestran resultados

Resultados del análisis del sistema de caracterización

En esta sección se muestran los resultados obtenidos a partir del análisis de la información ingresada por los estudiantes en el formulario de caracterización.

Los datos son procesados mediante un programa en Java, el cual permite clasificar a cada estudiante según su nivel de riesgo académico, teniendo en cuenta variables como el rendimiento académico, la situación socioeconómica y el estado emocional. La siguiente tabla presenta un ejemplo de cómo el sistema organiza la información de los estudiantes y muestra su nivel de riesgo:

En conclusión, el Sistema AURA permite recolectar, procesar y analizar información de los estudiantes con el fin de identificar posibles riesgos académicos y personales. Este prototipo integra conocimientos de análisis de requisitos, programación en Java y desarrollo web, mostrando una solución organizada y funcional. Muchas gracias.
